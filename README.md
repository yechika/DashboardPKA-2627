1. Buat file baru `.env.local`
2. Buka file `.env.local`, tambahkan :

```env
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=
MYSQL_PASSWORD=
MYSQL_DATABASE=
```

1. **Install Dependencies**  
   ```bash
   npm install
   ```

2. **Run**  
   ```bash
   npm run dev
   ```

3. **Open Browser**  
   **[http://localhost:3000](http://localhost:3000)**
